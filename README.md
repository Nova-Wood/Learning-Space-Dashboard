# 🎓 专属科研与日常管理系统 (Learning Space Dashboard)

欢迎来到我的开源科研打卡系统！这是一个基于纯 Python (Streamlit) 打造的轻量级、本地化科研手账与效率看板。

**告别纸质手账和复杂的效率软件，用数据量化你的硕士/博士生涯！**

## ✨ 核心功能
* **⏰ 沉浸式打卡**：分时段（上午/下午/晚上/深夜）、分地点（工位/图书馆等）实时签到签退。
* **📚 阅读计划**：专属文献库管理，支持记录阅读进度与笔记。
* **📋 任务看板**：零碎事项管理，一键勾选完成，告别遗忘。
* **📊 数据洞察**：自动生成每日时长趋势图、精力分布饼图、心情统计。
* **⚙️ 高度定制**：在网页左侧边栏可随时修改系统名称和今日目标（打气语录！）。

---

## 🚀 快速上手教程

本系统完全在本地运行，数据保存在你自己的电脑里（自动生成 `.csv` 文件），绝对保护隐私。

### 第一步：准备基础环境
你需要电脑里装有 **Python**。
1. 前往 [Python 官网](https://www.python.org/downloads/) 下载并安装最新版。
2. ⚠️ **巨坑警告**：安装时，**务必勾选界面底部的 `Add Python.exe to PATH`**（将 Python 添加到环境变量），然后再一直点下一步直到完成。

### 第二步：下载本项目代码
你有两种方式获取代码：
* **方式 A (推荐懂 Git 的同学)**：在终端运行 `git clone https://github.com/你的用户名/你的仓库名.git`
* **方式 B (小白友好)**：点击本网页右上角的绿色按钮 **`<> Code`**，选择 **`Download ZIP`**，下载后解压到你电脑里的常用文件夹（比如桌面）。

### 第三步：安装依赖库
1. 打开你电脑的终端（Windows 按 `Win+R` 输入 `cmd` 回车；Mac 打开“终端”应用）。
2. 使用 `cd` 命令进入你刚刚解压代码的文件夹。例如：
   ```bash
   cd Desktop/Master-Learning-Space
3. 运行以下命令，一键安装所需的第三方包：
Bash
pip install -r requirements.txt
(如果下载很慢，可以在命令后面加上镜像源，如：-i https://pypi.tuna.tsinghua.edu.cn/simple)

### 第四步：启动系统！
环境配置好后，以后每次想要打卡，只需要在代码所在的文件夹打开终端，输入：

Bash
streamlit run app.py
你的默认浏览器会自动弹出一个网页，这就是你的专属科研基地了！

## 💡 进阶玩法：如何优雅地“双击秒开”？
如果你觉得每次都要敲代码太麻烦，可以制作一个本地启动器：

对于 Windows 用户：

在代码文件夹里，新建一个文本文档（.txt）。

里面只写一行代码：streamlit run app.py。

保存后，将这个文件的后缀名改为 .bat（例如命名为 启动打卡系统.bat）。

以后每天到工位，只需要双击这个 .bat 文件，系统就会自动在后台运行并打开网页啦！

对于 Mac 用户：

新建一个纯文本文件，写入 streamlit run app.py，保存为 run.command。

打开终端，运行 chmod +x /路径/到/你的/run.command 赋予执行权限。

以后双击该文件即可运行。
